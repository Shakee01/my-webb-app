# My Azure Web App – ASP.NET Core

This project is a simple web application created using ASP.NET Core and deployed to Microsoft Azure App Service.

## Description

The application was created as part of a learning exercise about deploying web applications to the cloud.

It is a minimal ASP.NET Core (.NET 8) application that runs on Azure and displays a simple message page.

## Live Demo

You can view the deployed application here:

https://patryk-webapp-demo-ehgugkh7gqenfhc6.westeurope-01.azurewebsites.net/

## Technologies

* ASP.NET Core (.NET 8)
* C#
* Microsoft Azure App Service
* GitHub Actions (CI/CD)

## Project Structure

```
my-webb-app

 └ .github/workflows
```

## Purpose

The goal of this project was to learn:

* how to create a minimal ASP.NET Core application
* how to deploy an application to Azure
* how to use GitHub Actions for automatic deployment
